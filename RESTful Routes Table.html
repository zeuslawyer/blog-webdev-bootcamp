RESTful Routes

A table of all 7 RESTful routes

Name Path HTTP Verb Purpose Mongoose Method
Index /[items] GET List all [items] Item.find()
New /[items]/new GET Show new [item] form N/A
Create /[items] POST Create a new [item], then redirect somewhere Item.create()
Show /[items]/:id GET Show info about one specific [item] Item.findById()
Edit /[items]/:id/edit GET Show edit form for one [item] Item.findById()
Update /[items]/:id PUT Update particular [item], then redirect somewhere Item.findByIdAndUpdate()
Destroy /[items]/:id DELETE Delete a particular [item], then redirect somewhere Item.findByIdAndRemove()
